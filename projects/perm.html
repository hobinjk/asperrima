Perm

Perm is currently an AES-256 encryption breaker. It uses a brute force dictionary attack with a twist. Instead of just running through lowercase or uppercase words, it also checks the l33t-speak permutations. For example, here are some of the permutations of "example":

example3xampleex4mple3x4mpleexamp|e
3xamp|eex4mp|e3x4mp|eexamp1e3xamp1e
ex4mp1e3x4mp1eexamp!e3xamp!eex4mp!e
3x4mp!eexampl33xampl3ex4mpl33x4mpl3

Why l33tspeak?

In this age of password security, simple passwords, like admin, itstaff, or even password are considered secure if some letters are replaced with their l33t equivalents, e.g. adm1n, itst4ff, or p4$$w0rd. However, this program demonstrates that this is not the case. In about 3 minutes on current hardware, Perm is able to run through the l33tspeak and switchcased versions of every word in the English language. The methods of selecting passwords need to be rethought immediately for a more secure digital future.

Installation

Download the source archive. Install gcc and openssl if you haven't already. Figure out how to change the ciphertext it's working on, and compile with gcc perm.c leet.c -lcrypto. Note: this program is under development and the current posted version may provide false positives.
